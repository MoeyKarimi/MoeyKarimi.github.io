# StudyFlow Planner

StudyFlow Planner is a small student dashboard for planning study tasks, organizing subjects, and tracking weekly progress.

This project is designed to be a standalone portfolio project. Put these files in their own GitHub repository so visitors can download and run it separately from the main portfolio website.

## Features

- Add study tasks
- Choose subject and priority
- Mark tasks as complete
- Filter tasks by all, open, or done
- View total tasks, completed tasks, and progress percentage
- Save tasks in the browser with `localStorage`
- Reset demo data

## Tech Stack

- HTML
- CSS
- JavaScript

## How To Run

Download or clone the repository, extract the ZIP if needed, then open:

```text
index.html
```

No install step is required.

The app is self-contained in `index.html`, so it still runs even if someone only opens that file. The separate `styles.css` and `script.js` files are kept for easier editing.
