# InvoiceMate

InvoiceMate is a lightweight invoice generator for creating clean client invoices, calculating totals, and printing a polished invoice summary.

It is built as a standalone portfolio project and can run directly in the browser.

## Features

- Edit invoice number and date
- Add business and client details
- Add, edit, and remove line items
- Automatic subtotal, tax, and total calculations
- Print-ready invoice preview
- Reset demo data
- Self-contained `index.html`

## Tech Stack

- HTML
- CSS
- JavaScript

## How To Run

Download or clone the repository, extract the ZIP if needed, then open:

```text
index.html
```

No install step is required.
